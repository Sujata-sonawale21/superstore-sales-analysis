# Superstore Sales Analysis

## Project Overview

This project analyzes retail sales data from a global superstore using SQL and Power BI.

This project demonstrates an end-to-end data analytics workflow, including data validation, SQL analysis, KPI reporting, and interactive dashboard development.

The objective is to identify sales trends, product performance, regional performance, and customer purchasing behavior to support business decision-making.

---

## Tools Used

- SQL
- Power BI
- Excel

---

## Business Questions

1. Which product categories generate the highest sales?
2. Which regions contribute the most revenue?
3. Which products perform best?
4. How do sales trends change over time?
5. Which customer segment generates the highest sales?

---

## KPIs

- Total Sales
- Total Orders
- Total Customers
- Average Order Value (AOV)

---

## Dashboard Features

- Sales by Category

## SQL Analysis

The SQL analysis includes:

- Database Creation
- Table Creation
- Data Validation
- Business KPI Analysis
- Product Performance Analysis
- Geographic Analysis
- Customer Segment Analysis
- Time Series Analysis
- Shipping Analysis
- Power BI Views

---

## Key Insights

- Technology category generated the highest sales.
- West region contributed the highest revenue.
- Consumer segment generated the highest sales.
- Standard Class was the most used shipping mode.

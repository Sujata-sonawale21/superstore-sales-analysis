# Dataset Source

## Overview

This project uses the Superstore Sales Dataset for sales analysis and dashboard development.

The dataset contains sales transaction records including customer details, product information, order dates, shipping details, and sales values.

## Dataset Information

- Dataset Name: Superstore Sales Dataset
- Source: Kaggle
- Time Period: 2015 - 2018
- Records: Approximately 9,800 transactions

## Key Columns

- Order ID
- Order Date
- Ship Date
- Customer ID
- Segment
- Region
- Category
- Sub-Category
- Product Name
- Sales

## Dataset Link

https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting/data

## Note

The dataset is not included in this repository. It can be downloaded from the source link above.
